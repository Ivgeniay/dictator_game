# Общие типы

`Domain/Shared/`

Понятия предметной области которые используются во всех частях домена - в законах, в акторах, в симуляции. Не принадлежат ни законопроектам ни законам - они выше этого уровня.

## Constants

Все строковые константы домена собраны в одном месте. Организованы в категории через вложенные статические классы: `Subjects`, `Actions`, `Restrictions`, `Circumstances`, `All`, `Empty`.

Использование констант вместо строковых литералов гарантирует что одно и то же понятие всегда записывается одинаково - в `SubjectType`, в `DynamicMask`, в тестах и везде где используется строковое значение.

## LegalTerms

Типобезопасные обёртки над строками для юридических понятий языка законов. Все наследуют `StringType`.

`SubjectType` - кого затрагивает закон. `ActionType` - что регулируется. `RestrictionType` - насколько строго, имеет числовой уровень `Severity` для сравнения законов между собой. `CircumstanceType` - при каких обстоятельствах.

Каждый тип имеет набор статических значений по умолчанию и поддерживает создание новых значений динамически - например когда в процессе игры возникает новая группа населения.
