# Диктатор - Архитектурная документация

Это техническая документация архитектурного ядра игры "Диктатор" - симулятора авторитарного режима на C#.

Документация описывает **зачем существует каждая абстракция** и какую часть реального мира она моделирует. Это не справочник API.

## Структура проекта

```
Dictator.Domain      - доменный слой, чистые модели и абстракции
Dictator.Application - слой приложения, связи и взаимодействия
Dictator.TestConsole - консольный проект для быстрых тестов
```
