# Генератор случайных чисел

`Domain/Utils/RandomS.cs`

## RandomS

Статический потокобезопасный генератор случайных чисел. Единственный экземпляр `Random` на всё приложение защищённый через `lock`.

Использование единственного экземпляра вместо `new Random()` в каждом месте предотвращает ситуацию когда несколько генераторов созданных в одну миллисекунду возвращают одинаковые последовательности.

Используется везде в домене где нужна случайность - в `SinFactory`, в `PersonRelation.ApplyRandomEmotion`, в механиках расследования.
