# Сериализация

`Domain/Utils/Serd/`

Вспомогательные конвертеры для Newtonsoft.Json которые обеспечивают корректную сериализацию и десериализацию полиморфных иерархий.

## NodeJsonConverter

Конвертер для иерархии `Node`. При сериализации записывает объект как есть. При десериализации читает поле `Type` - полное имя класса - и через `typeof(Node).Assembly.GetType(typeName)` находит нужный конкретный тип в сборке `Dictator.Domain`. Использование `Assembly` вместо `Type.GetType` гарантирует корректную работу независимо от того в какой сборке вызывается конвертер.

## StringTypeJsonConverter

Конвертер для иерархии `StringType`. Работает по той же логике что `NodeJsonConverter` - читает `Type` и восстанавливает конкретный наследник. Необходим потому что `StringType` абстрактный и Newtonsoft не может создать его напрямую.

Наследники `StringType` имеют конструктор без параметров и свойства с `protected set` - это позволяет Newtonsoft заполнить поля после создания объекта через конструктор по умолчанию.
