# Маски

`Domain/Utils/Masks/`

Маски - это механизм определения принадлежности значения к некоторому множеству. В игре используются для ответа на вопрос "распространяется ли этот закон на этого субъекта".

## DynamicMask\<T\>

Динамическая маска включения/исключения. Моделирует флаговый enum но без ограничений на количество значений и с поддержкой динамического добавления новых.

Хранит полный список возможных значений переданных в конструктор и для каждого хранит состояние - включён или исключён. Оптимизирована для операции "включить всех" через флаг `_includesAll` - чтобы не перебирать весь словарь при проверке.

Используется когда нужно явно перечислить кого закон включает и кого исключает без учёта иерархии между субъектами.

## HierarchicalMask\<T\>

Иерархическая маска для типизированных строковых значений. Определяет вхождение субъекта в действие закона на основе его позиции в иерархии. Направление вхождения задаётся при создании: `Down` - закон распространяется на всех кто ниже или равен целевому уровню, `Up` - на всех кто выше или равен. Иерархия может изменяться динамически в процессе игры.

Моделирует реальность диктатуры где субъекты имеют разный уровень привилегий. Например закон для `People` де-факто не распространяется на силовиков и чиновников - они стоят выше в иерархии.

Поддерживает две перегрузки `Matches` - по значению и по числовому уровню - что позволяет сравнивать субъекта с уровнем напрямую без поиска в иерархии.
