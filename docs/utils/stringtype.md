# StringType

`Domain/Utils/StringType.cs`

Абстрактный базовый класс для типобезопасных строковых значений. Моделирует паттерн "строго типизированной строки" - когда строковое значение несёт конкретный доменный смысл и не должно смешиваться с произвольными строками.

## Зачем это нужно

В C# легко перепутать строки разного смысла - передать название субъекта туда где ожидается тип ограничения. `StringType` и его наследники делают такую ошибку невозможной на уровне компилятора.

Все наследники поддерживают сравнение по значению через переопределённые `Equals` и `GetHashCode`. Поле `Type` содержит полное имя конкретного класса - используется при сериализации для восстановления правильного типа.

## Наследники

`SubjectType`, `ActionType`, `RestrictionType`, `CircumstanceType` - доменные типы из `Shared/LegalTerms/`. Каждый из них имеет набор статических значений по умолчанию и поддерживает создание новых значений динамически в процессе игры.
