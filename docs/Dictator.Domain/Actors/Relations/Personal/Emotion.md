# Emotion

`Dictator.Domain\Actors\Relations\Personal\Emotion.cs`

## Emotion

*Класс*

Типобезопасная строка представляющая эмоцию которую личность испытывает к диктатору. Эмоции не исключают друг друга - страх и уважение могут существовать одновременно, что типично для условий авторитарного режима.
