# PersonRelation

`Dictator.Domain/Actors/Relations/Personal/PersonRelation.cs`

## PersonRelation

*Класс*

Личное отношение конкретного человека к диктатору. Выражается через набор активных эмоций и числовой уровень лояльности. Эмоции не исключают друг друга - человек может одновременно испытывать страх и уважение, что типично для условий диктатуры.
