# Sin

`Dictator.Domain\Actors\Relations\Personal\Sin.cs`

## Sin

*Класс*

Типобезопасная строка представляющая компрометирующий факт из жизни конкретной личности. Является материалом для шантажа, уголовного преследования и публикаций в СМИ.

## SinFactory

*Класс*

Фабрика для создания наборов грехов при генерации персонажей. Предоставляет методы для получения случайных неповторяющихся грехов - от одного до произвольного количества включительно. Используется при инициализации депутатов, министров и олигархов и других персонажей в отношении которых можно проводить расследования.
