# Person

`Dictator.Domain/Actors/Types/Person.cs`

## Person

*Класс*

Конкретный именованный представитель группы населения. Моделирует индивида с которым игрок может взаимодействовать напрямую - подкупать, шантажировать, запугивать. Хранит личные грехи которые могут быть раскрыты в ходе расследования и личное отношение к диктатору.
