# Loyalty

`Dictator.Domain/Shared/Metrics/Loyalty.cs`

## LoyaltyGrade

*Перечисление*

Качественная градация уровня лояльности — от открытой враждебности до безоговорочной преданности. Используется для определения поведения акторов без привязки к конкретным числам.

## Loyalty

*Класс*

Числовой уровень лояльности актора к диктатору. Динамически настраиваемый диапазон позволяет использовать класс для разных контекстов - личной лояльности персонажа, лояльности организации или группы населения.
