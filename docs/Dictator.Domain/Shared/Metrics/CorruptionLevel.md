# CorruptionLevel

`Dictator.Domain\Shared\Metrics\CorruptionLevel.cs`

## CorruptionGrade

*Перечисление*

Качественная градация уровня коррупции - от полного отсутствия до крайней степени. Используется для принятия решений в игровых механиках без привязки к конкретным числам.

## CorruptionLevel

*Класс*

Числовой уровень коррупции организации с динамически настраиваемым диапазоном. Влияет на эффективность расследований и вероятность саботажа действий диктатора.
