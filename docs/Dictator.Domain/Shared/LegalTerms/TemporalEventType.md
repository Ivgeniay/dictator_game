# TemporalEventType

`Dictator.Domain\Shared\LegalTerms\TemporalEventType.cs`

## TemporalEventType

*Класс*

Типобезопасная строка представляющая периодическое событие регулируемое законом. Используется в TemporalBillStatement для описания того что именно происходит через заданный интервал тиков — выборы, заседания, отчёты и т.д.
