# CircumstanceType

`Dictator.Domain\Shared\LegalTerms\CircumstanceType.cs`

## CircumstanceType

*Класс*

Типобезопасная строка представляющая обстоятельство при котором действует закон. Сужает область применения закона до конкретного места или контекста - в транспорте, в общественных местах, везде.
