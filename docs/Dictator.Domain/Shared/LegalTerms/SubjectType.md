# SubjectType

`Dictator.Domain\Shared\LegalTerms\SubjectType.cs`

## SubjectType

*Класс*

Типобезопасная строка представляющая категорию субъектов на которых распространяется закон. Поддерживает динамическое добавление новых категорий в процессе игры - когда возникают новые группы населения или институты.
