# RestrictionType

`Dictator.Domain/Shared/LegalTerms/RestrictionType.cs`

## RestrictionType

*Класс*

Типобезопасная строка представляющая степень ограничения действия законом. Числовой уровень строгости Severity позволяет программно сравнивать законы и вычислять направление изменения при внесении поправок.
