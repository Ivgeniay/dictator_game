# StringType

`Dictator.Domain\Utils\StringType.cs`

## StringType

*Класс*

Абстрактный базовый класс для типобезопасных строковых значений предметной области. Предотвращает случайное смешение строк разного смысла на уровне компилятора. Поле Type содержит полное имя конкретного класса и используется при сериализации.
