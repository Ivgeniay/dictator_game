# DynamicMask

`Dictator.Domain/Utils/Masks/DynamicMask.cs`

## DynamicMask

*Класс*

Динамическая маска включения и исключения для типизированных строковых значений. Моделирует флаговый enum без ограничений на количество значений и с поддержкой динамического расширения в процессе игры.
